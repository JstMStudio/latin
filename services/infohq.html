Colour

Users could also edit text colour in all InfoHQ. You could edit all text colour in InfoHQ pages and websites. By default, InfoHQ text colour is green with footer is blue. You could change it to any colour you likes if you want.

In order to change InfoHQ text colour, please insert this line to the tags you preferred which is provided below by us to change to any color you preferred. 

  
    style="color: red;"

Edit

All pages in InfoHQ is editable by everyone from admin to normal users and members. Users could edit all InfoHQ pages and websites like helping to add more information and etc. You could edit all pages of your InfoHQ by visiting the path: infohq /(InfoHQ database name)/ in the main directory of our website. Once you done, you could submit a pull request to wait for a pull request reviewer to approve it so it could go live on the InfoHQ websites.

Markdown

InfoHQ currently also support markdown on all InfoHQ websites. InfoHQ users and admin could use markdown on every InfoHQ websites and pages. You could use any types of markdown available too.

Basic Style

InfoHQ offered many basic styles in markdown. You could use most if not all basic style markdown in InfoHQ.

  • Strong : 
      
    strong Hi /strong
  • Italic :

    Installing

    In order to install InfoHQ, you firstly needed a web hosting that support hosting for static sites. Mostly free and paid web hosting provide support for static sites too. You could download InfoHQ in Github repository release tab. You could download any version you likes too here. After downloading, please upload to your hosting file section. Once you uploaded and created the file, edit the "Edit", "View Source" and "History" links and push it again. After the push succeed, your InfoHQ is now up and running for you to create and edit articles.
